# 本开源项目为PowerMill辅助插件  
## 主要目的是为入门开发者提供一个可以直接用来继续开发的插件框架，开发者可以使用本项目框架来进行重新开发以完成自己的辅助插件。  
## 此为[动力磨坊]Liu1ee独立开发项目。  
本项目使用Apache-2.0 license。  
本项目将阶段性更新。  

## 更新记录：

- 有小伙伴说如何将winform控件来开发垂直插件页面
